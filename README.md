# Programim në C++ – Klasa Telefoni
*Object-Oriented Programming Basics*

Mirë se vini në këtë repository!  
Ky projekt është krijuar për të ndihmuar **fillestarët** të kuptojnë bazat e **programimit në C++** dhe **programimit të orientuar në objekte (OOP)** duke përdorur një shembull praktik: **modelimin e një telefoni**.


## Përshkrimi i Projektit

Programi përmban një klasë të quajtur `telefoni` e cila ruan informacione bazë për një telefon dhe ofron funksione për përpunimin dhe shfaqjen e tyre.

Përdoruesi fut të dhënat nga tastiera dhe programi i shfaq rezultatet në ekran.



## Çfarë mësoni nga ky projekt?

- Si të krijoni një **class** në C++
- Si të përdorni **atribute private**
- Si të përdorni **konstruktorë**
- Si të krijoni dhe thërrisni **funksione**
- Si të punoni me `cin` dhe `cout`

Ky projekt është ideal për **studentë dhe fillestarë**.


## Çfarë përmban klasa `telefoni`

Klasa `telefoni` përmban këto të dhëna:

- Prodhuesi i telefonit
- Ngjyra
- Pesha
- Niveli i baterisë
- Memoria (në GB)
- Madhësia e ekranit (në inch)

Gjithashtu përmban funksione që:
- numërojnë shkronja në tekst
- gjejnë mesin e një fjale
- konvertojnë peshën në kilogram
- shfaqin nivelin e baterisë
- kthejnë informacionin e telefonit

## Udhëzime për të nisur një projekt lokal në GitHub

Këto hapa ju ndihmojnë të krijoni një projekt në kompjuterin tuaj dhe ta publikoni në GitHub.

### 1.Krijoni një repository në GitHub
- Hyni në faqen **github.com**
- Klikoni butonin **New repository**
- Vendosni emrin e projektit
- Klikoni **Create repository**


### 2.Krijoni projektin në kompjuter
- Krijoni një dosje të re në kompjuterin tuaj
- Vendosni brenda saj:
  - file-in e kodit (p.sh. `telefoni.cpp`)
  - file-in `README.md`


### 3.Hapni projektin në një editor
- Hapni dosjen e projektit në një editor (p.sh. Visual Studio Code)
- Sigurohuni që file-at janë hapur saktë

---

### 4.Lidheni projektin me GitHub
- Hapni seksionin **Source Control** në editor
- Inicializoni Git për projektin
- Shtoni file-at që dëshironi të ruani
- Shkruani një mesazh të shkurtër dhe ruani ndryshimet (commit)


### 5.Dërgoni projektin në GitHub
- Lidheni projektin me repository-n e krijuar në GitHub
- Dërgoni (push) ndryshimet
- Kontrolloni repository-n online për të parë projektin


### 6.Përditësimi i projektit
Sa herë që bëni ndryshime në projekt:
- Ruani ndryshimet
- Bëni commit
- Dërgoni përsëri në GitHub

**Këshillë**  
Mos u shqetësoni nëse diçka nuk funksionon menjëherë. GitHub mësohet hap pas hapi
